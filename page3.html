GitHub y VSCode

Los programas utilizados fueron GitHub para documentar lo estudiado en este archivo .html, mediante articulos de lectura y videotutoriales de YouTube pude aprender la finalidad de diferentes comandos de Git como git add, push, commit, fetch, branch, switch y pull. Los commits fueron propiamente agregados a medida que fueron haciendose adiciones y cambios a la pagina, todos en el branch "templado-final". El otro programa utilizado fue VSCode para hacer la pagina con lenguaje HTML y CSS, también con material de lectura y audiovisual pude aprender lo básico para montarme una página sencilla pero facil de utilizar.
